# 🔔 Telegram Бот-Напоминалка

Telegram-бот для создания и управления персональными напоминаниями.

**Стек:** Python 3.11+ · aiogram 3.x · SQLite (aiosqlite) · APScheduler · Pydantic Settings

---

## 📋 Что делает проект

Бот позволяет пользователю создавать напоминания с текстом и временем.  
В назначенное время бот отправляет сообщение в личный чат пользователя.

### Возможности

| Функция | Команда |
|--------|---------|
| Создать напоминание (пошаговый диалог) | `/add` |
| Просмотреть все активные напоминания | `/list` |
| Удалить напоминание (по id или через кнопку) | `/delete` |
| Помощь и описание команд | `/help` |
| Запустить / перезапустить бота | `/start` |

### Архитектурные решения

- **FSM (Finite State Machine)** — многошаговый диалог добавления напоминания через `aiogram.fsm`
- **Middleware** — инъекция соединения с БД в хендлеры через `DatabaseMiddleware`
- **APScheduler** — асинхронный планировщик, проверяет просроченные напоминания каждые N секунд
- **Pydantic Settings** — валидация и типизация переменных окружения
- **aiosqlite** — асинхронная работа с SQLite без блокировки event loop

## 🚀 Как запустить (пошагово)

### 1. Клонировать репозиторий

```bash
git clone https://github.com/your-username/telegram-reminder-bot.git
cd telegram-reminder-bot
```

### 2. Создать виртуальное окружение

```bash
# Создание
python -m venv .venv

# Активация на Linux/macOS
source .venv/bin/activate

# Активация на Windows
.venv\Scripts\activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Создать файл с переменными окружения

```bash
# Копируем шаблон
cp .env.example .env
```

Открываем файл `.env` и заполняем переменные:

```env
# Получить токен у @BotFather в Telegram
BOT_TOKEN=1234567890:AABBccDDeeFF-ваш_токен_здесь

# Остальные значения можно оставить по умолчанию
DATABASE_PATH=data/reminders.db
TIMEZONE=Europe/Moscow
LOG_LEVEL=INFO
SCHEDULER_INTERVAL=60
```

> **Как получить токен:**
> 1. Найдите в Telegram бота `@BotFather`
> 2. Отправьте команду `/newbot`
> 3. Следуйте инструкциям и скопируйте полученный токен

### 5. Запустить бота

```bash
python -m bot.main
```

Бот запустится и будет принимать сообщения. Для остановки нажмите `Ctrl+C`.

```

│   │   ├── start.py            # Хендлер /start

│   ├── keyboards/
│   │   └── inline.py           # Генераторы inline-клавиатур
│   ├── middlewares/
│   │   └── db_middleware.py    # Middleware для инъекции БД в хендлеры
│   └── scheduler/

├── tests/                      # Тесты
│   ├── conftest.py             # Фикстуры: тестовая БД, тестовые данные
│   ├── unit/                   # Юнит-тесты (изолированные)
│   │   ├── test_models.py      # Тесты модели Reminder
│   │   ├── test_config.py      # Тесты конфигурации
│   │   ├── test_crud.py        # Тесты CRUD-операций
│   │   └── test_keyboards.py   # Тесты клавиатур
│   └── integration/            # Интеграционные тесты

│   └── reminders.db            # Файл базы данных SQLite
├── .env                        # Переменные окружения (НЕ коммитится в git!)
├── .env.example                # Шаблон переменных окружения
├── .gitignore                  # Игнорируемые файлы для git
├── requirements.txt            # Зависимости Python
├── pyproject.toml              # Конфигурация проекта (PEP 517/518)
└── README.md                   # Этот файл
```

## 💬 Примеры использования

### Создание напоминания

```
Пользователь: /add

Бот: 📝 Шаг 1 из 2
     Введи текст напоминания.

Пользователь: Позвонить врачу

Бот: 🕐 Шаг 2 из 2
     Введи дату и время в формате:
     ДД.ММ.ГГГГ ЧЧ:ММ
     Например: 25.12.2024 09:30

Пользователь: 25.12.2024 09:30

Бот: ✅ Проверь данные напоминания:
     📝 Текст: Позвонить врачу
     🕐 Время: 25.12.2024 09:30
     Всё верно?
     [✅ Сохранить] [❌ Отмена]

Пользователь: (нажимает «Сохранить»)

Бот: ✅ Напоминание сохранено!
     📝 Позвонить врачу
     🕐 Напомню 25.12.2024 09:30
```

### Просмотр списка

```
Пользователь: /list

Бот: 📋 Твои напоминания (2):

     1. [#1] 25.12.2024 09:30 — Позвонить врачу
     2. [#2] 31.12.2024 23:00 — Купить шампанское

     [🗑 Удалить #1]
     [🗑 Удалить #2]
```

### В назначенное время

```
Бот: 🔔 Напоминание!

     Позвонить врачу
```
